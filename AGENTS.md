## Design

This project ships a `DESIGN.md` (Gesso grammar + skin) and
`design.tokens.css`. Read `DESIGN.md` before building or changing any UI,
and use the CSS custom properties from `design.tokens.css` for all colors,
type, spacing, radius, and motion. Do not hardcode values that exist as
tokens. Validate with: `npx --yes @google/design.md lint DESIGN.md`.
